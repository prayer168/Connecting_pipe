# 連通管探險站：水面會停在哪裡？

給國小中年級至高年級使用的 RWD 數位互動教材。新版參考「毛細現象探究實驗室」的分頁式版面，讓學生依序完成觀察、操作、圖表、原理、生活應用與測驗。

## 開啟教材

- 線上教材：<https://prayer168.github.io/Connecting_pipe/>
- 本地使用：直接以瀏覽器開啟 `index.html`

## 內容

- `觀察站`：連通管情境導入與水面等高動畫。
- `互動實驗室`：切換杯型、注水量、倒水位置、觀察速度、水平提示線與減少動態模式。
- `我的圖表`：比較一開始與平衡後的水面高度，可帶入剛剛的實驗結果。
- `原理小教室`：以 canvas 圖解底部相通、壓力平衡與杯型無關。
- `生活中的應用`：相通透明杯、水準管、水位管、茶壺、水塔與 U 形管案例。
- `挑戰測驗`：四題選擇題與即時回饋。
- 適用桌機投影、平板與手機的 RWD 版面。

## 檔案結構

- `index.html`：教材頁面結構。
- `styles.css`：版面、RWD 與視覺樣式。
- `script.js`：分頁、模擬、圖表、生活應用與測驗互動。
- `assets/previews/`：桌機與手機預覽圖。

## 教學範圍

第一版呈現清水、容器底部相通且上方開放的情境。動畫速度為教學示意，尚未包含不同密度液體、外加壓力、毛細作用或真實流速計算。
